# @maxun/core

Core utilities for Maxun SDKs. This package is intended for internal use by `@maxun/extract` and `@maxun/scrape`.

## Installation

```bash
npm install @maxun/core
```

## What's Included

- `MaxunClient` - API client for communicating with Maxun backend
- `WorkflowBuilder` - Base class for building workflows
- `Robot` - Robot management and execution
- Type definitions for the Maxun platform

## Usage

This package is typically not used directly. Instead, use `@maxun/extract` or `@maxun/scrape` which build on top of this core functionality.
